# Python GUI Application Project

This is a test project designed for developing a Graphical User Interface (GUI) application using Python.

## 🚀 Project Goal
The primary goal of this project is to build and test a functional desktop GUI application.

## ⚙️ Technical Specifications & Environment
*   **Target Hardware:** AMD RX Vega56 8GB GPU.
*   **Development Tooling:** LM Studio for local model management/testing.
*   **Status:** Test Project / Proof of Concept.

## 💡 Model Integration Status
Please note that this current iteration is **not** integrated with any specific large language model (LLM), such as google/gemma-4-e4b. The focus remains on developing and testing the core GUI functionality.
